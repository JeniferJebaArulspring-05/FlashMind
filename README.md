🧠 FlashMind

A simple and interactive flashcard quiz Android application built using Kotlin and Jetpack Compose to help users learn and revise concepts easily.

✨ Features
📚 Flashcard-based learning system
🔁 Next / Previous navigation
👀 Show / Hide answer option
🎨 Beautiful UI with background image
🟣 Modern purple themed flashcards
📱 Fully responsive Android app
🛠️ Tech Stack
Kotlin
Jetpack Compose
Material 3
Android Studio
